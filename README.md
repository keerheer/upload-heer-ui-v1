# heer-ui

## 项目介绍
基于 Vue 3 Composition API 开发的组件库
## 项目文档
基于 VitePress 构建的项目文档
下载完成后 进入项目目录

```
bash
cd docs
npm run docs:dev    
```

## 下载项目-项目创建
```
yarn install
```

### 启动项目
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
